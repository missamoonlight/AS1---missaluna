# Missa Luna - ArtistPage

Landing page para el artista independiente Missa Luna, diseñada para mostrar sus últimos lanzamientos musicales a través de reproductores de SoundCloud integrados.

## Descripción

Este proyecto es una página web responsiva que funciona como carta de presentación de la artista. Incluye un diseño adaptable a dispositivos móviles, sistema de cambio entre modo claro y oscuro, y una sección interactiva con JavaScript.

## Características

- Barra de navegación fija con enlaces a todas las secciones
- Hero section con imagen de la artista y llamado a la acción
- Galería con cuatro reproductores de SoundCloud
- Botón interactivo que cambia mensajes y estilos
- Sistema de modo oscuro con actualización dinámica de iconos y textos
- Pie de página con información de contacto y enlaces

## Tecnologías

- HTML5
- CSS3 (Bootstrap 5.3, Bootstrap Icons, estilos personalizados)
- JavaScript (Vanilla)
